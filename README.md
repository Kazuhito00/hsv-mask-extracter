# hsv-mask-extracter
 hsv-mask-extracterはHSV値を閾値にしたマスク画像を生成するプログラムです。
 
# Requirement
 
* OpenCV 3.4.2(or later)
 
# Installation
 
ディレクトリを丸ごとコピーして実行してください。
 
# Usage
 
サンプルの実行方法は以下です。
 
```bash
python hsv_mask_extracter.py
```
